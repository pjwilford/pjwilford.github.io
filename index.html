HTML Cheatsheet

Selector Syntax Description
!DOCTYPE <!DOCTYPE> Places standard set-up for your HTML documentation
html <html> between the opening and closing tags, this is where you write your html code for your site build.
link <link> When you want to call in information from another file or directory in your project, you can link to it using this element.

CSS Cheatsheet

Property Syntax Outcome
* * {} Affects styling "universally," aka will affect the entire document.
body body {} Affects styling only for the html code written in the <body> element
.(class) .(class) {} Affects styling only for the named class ("class" is changed to whatever you've titled the class attribute of the element you're trying to style.) Can affect one or multiple items - dependant on the class name and how many have that same class name.

There are many more selectors and properties that can be used with HTML and CSS. Over time, this list can be added to using the tools and techniques learned on Codecademy.com.
